## 疑似要素で実現する追加マークアップ不要のデザイン

`::before` と `::after` は、HTML を追加することなく装飾要素を挿入できる強力なツールです。単なる「矢印アイコンの追加」だけではなく、複雑な装飾パターン、アニメーション効果、レイアウト補助など、多様な用途に活用できます。

疑似要素の真の価値は、「構造」と「装飾」を分離できることにあります。HTML は意味的な構造のみを保ち、視覚的な装飾は CSS に任せることで、保守性が劇的に向上します。また、JavaScript を使わずに動的な視覚効果を実装できるため、パフォーマンスにも優れています。

このガイドでは、実務で使える疑似要素のクリエイティブなテクニックを、実際に動くデモとともに紹介します。

}

```css
  content: '🔥';
  margin-right: 0.5rem;
```

### 高度な疑似要素テクニック

```css
/* カウンター機能 */
.list {
  counter-reset: item; /* カウンターを初期化 */
}

.list-item::before {
  content: counter(item) '. '; /* カウンターを表示 */
  counter-increment: item; /* カウンターをインクリメント */
}

/* data属性の値を表示 */
.tooltip::before {
  content: attr(data-tooltip);
}

/* 三角形を作る */
.arrow::after {
  content: '';
  width: 0;
  height: 0;
  border-style: solid;
  border-width: 10px 10px 0 10px;
  border-color: #60a5fa transparent transparent transparent;
}
```

## Tips & Gotchas

### ✅ 実践的なテクニック

1. **content は必須**
   ```css
   /* ❌ 表示されない */
   .element::before {
     background: red;
   }

   /* ✅ 正しい */
   .element::before {
     content: ''; /* 空でも必須 */
     background: red;
   }
   ```

2. **z-index の活用**
   ```css
   .element {
     position: relative;
     z-index: 1; /* コンテンツを前面に */
   }

   .element::before {
     position: absolute;
     z-index: -1; /* 背景として配置 */
   }
   ```

3. **複数の疑似要素が必要な場合**
   ```css
   /* ラッパー要素を追加 */
   <div class="wrapper">
     <div class="content"></div>
   </div>

   .wrapper::before,
   .wrapper::after,
   .content::before,
   .content::after {
     /* 最大4つの疑似要素 */
   }
   ```

4. **アクセシビリティの考慮**
   ```css
   /* 装飾的な content は読み上げられない */
   .icon::before {
     content: '🔥';
     aria-hidden: true; /* スクリーンリーダーから隠す */
   }

   /* 意味のあるコンテンツは HTML に記述 */
   ```

### ⚠️ 注意点

1. **input や img には使えない**
   ```html
   <!-- ❌ 動作しない -->
   <input type="text" class="has-before">
   <img src="..." class="has-after">

   <!-- ✅ ラッパーを使用 -->
   <div class="input-wrapper">
     <input type="text">
   </div>
   ```

2. **position: absolute の基準点**
   ```css
   .element {
     position: relative; /* 親要素に relative を指定 */
   }

   .element::before {
     position: absolute;
     top: 0;
     left: 0; /* .element を基準に配置 */
   }
   ```

3. **パフォーマンス**
   ```css
   /* ⚠️ 大量の疑似要素は避ける */
   /* 特にアニメーションさせる場合は注意 */

   /* ✅ transform と opacity のみでアニメーション */
   .element::before {
     transition: transform 0.3s, opacity 0.3s;
   }
   ```

### 4. クリアフィックス

```css
  display: table;
  clear: both;
}
```

## まとめ

疑似要素の活用により:

- **HTML の簡潔化**: 装飾用のマークアップが不要
- **保守性の向上**: 構造と装飾の分離
- **パフォーマンス**: JavaScript 不要の視覚効果
- **柔軟性**: CSS のみで多様なデザインパターンを実装

`::before` と `::after` を使いこなすことで、クリーンで保守性の高いコードを書けるようになります。
